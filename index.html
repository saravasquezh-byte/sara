introducción

Un lenguaje de programación es un sistema de instrucciones y reglas que permite comunicarse con una computadora para decirle qué hacer. Con un lenguaje de programación se crean aplicaciones, páginas web, videojuegos, inteligencia artificial, sistemas operativos y mucho más.

LOGICA DE programacion

La lógica de programación es la manera de pensar y organizar pasos para resolver problemas usando una computadora. Consiste en crear instrucciones en un orden correcto para que la máquina realice una tarea, como sumar números, tomar decisiones o repetir acciones. La lógica incluye conceptos básicos como variables, condiciones, ciclos y funciones, y es la base de cualquier lenguaje de programación. Primero se piensa la solución mediante algoritmos o pseudocódigo y luego se escribe en un lenguaje como Python, Java o JavaScript.

TIPOS DE DATOS

Los tipos de datos son las diferentes clases de información que puede manejar un programa. Los más comunes son: entero, que guarda números sin decimales; flotante o decimal, que almacena números con decimales; cadena de texto, que contiene palabras o caracteres; booleano, que solo puede tener los valores verdadero o falso; y carácter, que guarda un único símbolo o letra.

OPERADORES

Los operadores aritméticos son símbolos usados para realizar operaciones matemáticas como suma (+), resta (-), multiplicación (*), división (/), módulo (%) y potencia (**). Los operadores lógicos se utilizan para combinar o evaluar condiciones y son: AND (y), OR (o) y NOT (no), permitiendo determinar si una expresión es verdadera o falsa.

CONTINUACION

CONDICIONALES

Las condicionales son estructuras de programación que permiten ejecutar diferentes acciones según se cumpla o no una condición. La instrucción if se utiliza para ejecutar un bloque de código cuando una condición es verdadera, else ifpermite comprobar otra condición diferente si la primera no se cumple, y else se usa para ejecutar una acción cuando ninguna de las condiciones anteriores es verdadera. Estas estructuras ayudan a que el programa tome decisiones y responda a distintas situaciones.

CONTINUACION

SWTCH CASE

La estructura switch case es una condicional que permite comparar una variable con varios valores posibles y ejecutar diferentes bloques de código según el caso. case define cada valor que se quiere evaluar; si la variable coincide con ese valor, se ejecuta el código de ese caso. break se usa para detener la ejecución del switch una vez que se ha encontrado y ejecutado un caso, evitando que siga revisando los demás. default es la opción que se ejecuta cuando ningún case coincide con el valor de la variable, funcionando como una respuesta por defecto

WHILE

El while es una estructura de repetición (ciclo) que ejecuta un bloque de código mientras una condición sea verdadera. Antes de cada repetición, el programa verifica la condición, y si sigue siendo verdadera, repite el código; cuando la condición se vuelve falsa, el ciclo termina. Es útil cuando no se sabe exactamente cuántas veces se repetirá una acción.
